# OMNI-NEXUS v13.0 | Bio-Performance Console
**Sistema de Monitorización Biomecánica para Atletas de Alto Rendimiento.**

Este sistema ha sido diseñado para la optimización de la hipertrofia mediante el análisis de variables críticas:
- **e1RM Dinámico:** Cálculo algorítmico de la fuerza máxima teórica.
- **Registro Fenomenológico:** Captura de sensaciones y fatiga central mediante RPE/RIR.
- **Arquitectura PWA:** Ejecución de baja latencia con persistencia de datos en LocalStorage.

## Instrucciones de Instalación
Para una experiencia de 60fps, abra el enlace en su navegador móvil y seleccione **"Añadir a pantalla de inicio"**.
